MANUAL DO SISTEMA

IFSP – Campus Presidente Epitácio

Sistema de Gerenciamento de REDs

1. Introdução

Este manual tem como objetivo orientar os usuários na utilização do sistema de gerenciamento de REDs, permitindo a navegação e execução das principais funcionalidades.

2. Funcionalidades do Sistema

2.1 Cadastro de Disciplinas

Permite cadastrar novas disciplinas no sistema, informando os dados necessários para sua identificação e associação.

2.2 Gerenciamento de Servidores

Possibilita o cadastro, edição e visualização dos servidores da instituição.

2.3 Processos RED

Permite criar, visualizar e acompanhar os processos de RED dentro do sistema.

2.4 Gerenciamento de PEE

Área responsável pela criação e acompanhamento dos Planos de Ensino Especiais.

3. Fluxo do Processo de RED

O processo de RED no sistema segue uma sequência definida entre os setores envolvidos:

1. CRA cria o RED
2. Coordenador aprova / confirma
3. CSP realiza a associação de disciplinas
4. Coordenador indica os docentes
5. Docentes preenchem os documentos iniciais
6. Aluno realiza as atividades
7. Docentes preenchem os documentos finais
8. CSP finaliza o processo
9. CRA gera o relatório final para envio ao SUAP
