# Sistema de Gerenciamento Regime Exercício Domiciliar - 2023

## Escopo do sistema
O Sistema de Gerenciamento do Regime de Exercícios Domiciliares (GRED) tem como objetivo primordial automatizar e organizar os processos relacionados à concessão, acompanhamento e registro do regime de exercícios domiciliares para os estudantes, conforme as diretrizes do Regime de Exercícios Domiciliares constituída no Decreto-lei n° 1.044/1969.

O GRED proporcionará maior eficiência, transparência e agilidade na administração deste regime, beneficiando tanto os estudantes quanto a instituição de ensino.
